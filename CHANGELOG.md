## Migration Guide

### From v8.0 to v10.0

1. **Update repository:**
   ```bash
   git pull origin main
   ```

2. **Run setup:**
   ```bash
   ./scripts/setup.sh
   ```

3. **Choose platform:**
   - CLI: `cd cli && python main.py`
   - Web: `./scripts/run-dev.sh`
   - Docker: `docker-compose -f docker/docker-compose.yml up -d`

4. **Configure environment:**
   - Copy `.env.example` to `.env` in each platform
   - Update with your settings
