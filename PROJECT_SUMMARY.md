## 🚀 How to Run

### Web Dashboard

1. **Navigate to frontend directory**
```bash
cd frontend
```

2. **Install dependencies** (if not already done)
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Access the application**
```
http://localhost:5173/
```

5. **Build for production**
```bash
npm run build
```
