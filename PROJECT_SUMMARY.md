### Quick Start
```bash

# Run the tool

# Follow the interactive menu
```

1. **Start the tool** → Beautiful banner appears
2. **Select option 1** → Start SMS Bombing
3. **Accept terms** → Type 'yes'
4. **Enter phone number** → 10-digit Indian number
5. **Enter wave count** → 1-100 waves

7. **Confirm** → Type 'yes'
8. **Watch the magic** → Real-time bombing with stats
